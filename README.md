# L-BOX

ローカルフォルダ内の音源をクロスフェードでつないで流す、ブラウザ完結型のランダムジュークボックスです。
ウェブサイトにアクセスし、お手持ちの音楽フォルダを選択するだけで動作します。

## 主な機能

- **クロスフェード再生**: 曲の切り替わりを滑らかにつなぎます。
- **揺らぎエフェクト**: 再生速度を微小に変化させ、心地よいリスニング環境を作ります。
- **スリープタイマー**: 指定した時間で自動的に再生を停止します。
- **ロック画面・通知パネル操作**: PWA技術とMedia Session APIにより、バックグラウンド再生や端末のロック画面からの曲送り操作に対応しています。
- **安心のプライバシー**: 選択した音楽ファイルやフォルダのデータが、外部のサーバーにアップロードされることは一切ありません。すべての処理はブラウザ内（ローカル環境）で完結します。

## 使い方

1. 公開されているURLにブラウザでアクセスします。
2. 画面の指示に従って、音楽ファイルが保存されているローカルフォルダを選択します。
3. 再生ボタンを押すと、ランダムに楽曲の再生が始まります。

※PWAとして端末にインストールして使用する場合は、HTTPSで保護された通信環境からアクセスする必要があります。
